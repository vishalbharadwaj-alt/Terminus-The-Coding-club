## Pages

- **Home** (`index.html`) - Landing page with hero section
- **About** (`about.html`) - Club information and mission
- **Events** (`events.html`) - Upcoming events and workshops
- **Projects** (`projects.html`) - Featured club projects
- **Join** (`join.html`) - Membership application form

## Local Development

```bash
# Install dependencies
npm install

# Start local server
npm run serve

# Visit http://localhost:5173
```

## Deployment

This site is ready to deploy to:

- **GitHub Pages** - See [GITHUB_PAGES_SETUP.md](./GITHUB_PAGES_SETUP.md)
- **Netlify** - Drag and drop the `public` folder
- **Vercel** - Connect your Git repository
- **Cloudflare Pages** - Upload or connect Git

See [DEPLOY.md](./DEPLOY.md) for detailed deployment instructions.
